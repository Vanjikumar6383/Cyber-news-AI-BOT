🛡️ Cyber News – Real-Time Cyber Intelligence Platform
📌 Overview

Cyber News is a futuristic cybersecurity news platform that delivers real-time cyber threat intelligence from trusted sources. It features a hacker-themed UI with interactive 3D animations and severity-based news classification.

🚀 Features
🔥 3-second animated intro (Three.js + GSAP)
📰 Real-time cyber news fetching (RSS feeds)
🌍 Country-based filtering
🚨 Severity ranking (High / Medium / Low)
💾 Save news (wishlist feature using localStorage)
🔄 Auto-refresh on reload
🎨 Neon hacker-style UI with animations
🛠️ Tech Stack

Frontend:

HTML, CSS, JavaScript
TailwindCSS
Three.js (3D animations)
GSAP (animations)

Backend (Planned / Optional):

Node.js + Express
MongoDB
📡 Data Sources
The Hacker News
BleepingComputer
Dark Reading
Krebs on Security

⚙️ How It Works
Fetches cybersecurity news from multiple RSS feeds
Converts RSS → JSON using API
Classifies news based on keywords (severity detection)
Filters by country and threat level
Displays in an interactive dashboard
